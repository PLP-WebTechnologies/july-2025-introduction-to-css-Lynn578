Understanding the Box Model

Every element in web design is a rectangular box. The CSS Box Model describes how these boxes are sized and spaced.

Margin

The outer layer that creates space between elements.

Border

The edge surrounding the padding and content.

Padding

The inner layer that creates space inside the element.
